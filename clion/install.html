Instalace CLion

Pro Debian/Ubuntu

sudo snap install clion

Pro Windows

Stáhněte instalační balík ze stránky https://www.jetbrains.com/clion/ po kliknutí na "GET FREE 30 DAY TRIAL".

Spousťte stažený EXE soubor
